To run example UI using current Viem based version of wagmi:

```
pnpm install

cd examples/wagmi-viem-example
pnpm install
pnpm dev
```

To run example UI using legacy Ethers based version of wagmi:

```
pnpm install

cd examples/wagmi-ethers-example
pnpm install
pnpm dev
```
